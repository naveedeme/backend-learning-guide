## 🚀 Deploy Your Own Copy

### Method 1 — Fork + GitHub Pages (Easiest)

1. **Fork** this repository
2. Go to your fork → **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. Push any commit to trigger the deploy workflow
5. Visit `https://YOUR_USERNAME.github.io/fullstack-learning`

### Method 2 — Clone and run locally

```bash
git clone https://github.com/naveedeme/fullstack-learning.git
cd fullstack-learning

# Serve locally (any static server works)
npx serve .
# or
python3 -m http.server 8080
```

Then open `http://localhost:8080`

## 📱 PWA Installation

The app works offline and can be installed as a native app:

- **Desktop Chrome/Edge**: Click the install icon in the address bar
- **Android Chrome**: Tap "Add to Home Screen" in the browser menu
- **iOS Safari**: Tap Share → "Add to Home Screen"
